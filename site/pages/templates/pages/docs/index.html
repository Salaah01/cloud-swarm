Documentation

1. Create Account

The very first step you need to take in order to get started is to create an account.

2. Register a Site

Add a new site to your account by clicking the "Add Site" button on the Sites page.

Complete the form entering the details requested and submit.

3. Update DNS Settings

Once you have completed the site registration form, you will be directed to your site dashboard.

Below is an example of what we should see assuming we had registered Bluish Pink in the previous step:

Couple things to notice:

      1. Site is unverified
      2. A random set of characters
      3. A verify site button

Our site is intended to be used for testing purposes only. However, we do see the potential for abuse. Therefore, we require that you verify your site before you are able to start using our services.

In order to verify your site, you will need to update your DNS settings and create a TXT record with the key provided by us (point 2).

Using the example above, we would create a TXT record with the following settings:

Type: TXT

Hostname: @

Value: cloud-swarm-verification=4c972f6b7b8d47255bb078a6230d0edf5dd25b1e757eacdd31b7370f3e53b3a6

Once you have created the TXT record, you will be able to verify your site by clicking the "Verify Site" button.

4. Run Benchmark

Click on the "New Benchmark" button to start a new benchmark. You will be directed to a page where you can select your benchmark parameters.

Number of Server: The number of servers you want to run the benchmark on. Each server would have its own IP address. This is an important feature we provide as some servers can throttle requests coming from the same IP address.

Number of Requests: The number of requests you want to send to from each server.

Once you are ready, click on "Submit" and you will be directed back to the site page from where you will be able to monitor the benchmark.
